# План автоматизации тестирования — OldBoy Barbershop + Akita

> **Цель:** Автоматизировать smoke-тесты главной страницы https://oldboybarbershop.com с помощью Akita
> **Инструмент:** Akita (BDD library, Cucumber + Selenide)
> **Дата:** 2026-07-01

---

## Общая последовательность действий

### Фаза 1: Подготовка окружения

- [x] Установить Java 8
- [x] Установить плагины IDEA: Cucumber for Java, Gherkin, Lombok
- [ ] Настроить Lombok (Enable annotation processing)
- [ ] Скачать WebDriver (ChromeDriver) и прописать в PATH

### Фаза 2: Настройка проекта

- [x] Создать структуру проекта на основе akita-testing-template
- [x] Настроить `build.gradle` с зависимостью `akita:4.0.0`
- [x] Настроить `application.properties` (URL сайта, таймауты)
- [x] Настроить `gradle.properties` (браузер по умолчанию)

### Фаза 3: Создание Page Objects

- [ ] Создать `pages/MainPage.java` — главная страница
  - [ ] Локаторы: Hero-заголовок, меню навигации, CTA-кнопки
  - [ ] Локаторы: секция услуг, карточки услуг
  - [ ] Локаторы: блок мобильного приложения, блок косметики
  - [ ] Локаторы: карта филиалов, блок франшизы, footer

### Фаза 4: Создание пользовательских шагов (Steps)

- [ ] Создать `steps/MainPageSteps.java` — кастомные шаги для проверок главной страницы

### Фаза 5: Создание feature-файлов (Gherkin)

- [ ] Создать `features/smoke.feature` — сценарии из TEST_CASES.md
  - [ ] TC-1: Загрузка главной страницы
  - [ ] TC-2: Отображение навигационного меню
  - [ ] TC-3: Отображение Hero-блока
  - [ ] TC-4: Отображение блока услуг
  - [ ] TC-5: Отображение блока мобильного приложения
  - [ ] TC-6: Отображение блока косметики
  - [ ] TC-7: Отображение карты филиалов и франшизы
  - [ ] TC-8: Отображение Footer

### Фаза 6: Запуск и отладка

- [ ] Запустить тесты: `./gradlew clean test -Pbrowser=chrome`
- [ ] Проанализировать результаты
- [ ] Сгенерировать отчёт: `./gradlew generateCucumberReport`
- [ ] Исправить ошибки при необходимости

### Фаза 7: Формирование отчёта

- [ ] Сохранить отчёт Cucumber
- [ ] Приложить результаты к карточке участника в Kaiten

## Структура создаваемых файлов

```
barbershop-akita-test/
├── build.gradle
├── settings.gradle
├── gradle.properties
├── src/
│   └── main/
│       └── java/
│           └── pages/
│               └── MainPage.java          ← Page Object главной страницы
|               └── steps/
|                   └── CustomSteps.java   ← Пользовательские шаги
│       └── resources/
│           └── application.properties
│   └── test/
│       └── resources/
│           └── features/
│               └── smoke.feature          ← Smoke-тесты (Gherkin)
```
