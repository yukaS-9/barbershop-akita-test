## TC-1: Загрузка главной страницы

**Проверка:** Главная страница успешно загружается и отображает базовые элементы.

```gherkin
Функционал: Главная страница OldBoy Barbershop
  Сценарий: Загрузка главной страницы
    Дано совершен переход на страницу "Главная" по ссылке "mainPageUrl"
    Тогда страница "Главная" загрузилась
    И в заголовке страницы присутствует текст "OldBoy Barbershop"
```
